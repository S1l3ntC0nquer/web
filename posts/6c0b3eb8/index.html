Fundamentals of Logic

Statements

  • Statements (or propositions)
    • Declarative sentences that are either true or false but not both
  • Primitive statements
    • There is really no way to break them down into anything simpler

New statements can be obtained from existing ones in two ways

  • Negation
    • We do not consider the negation of a primitive statement to be a primitive statement
    • The negation statement of ppp is ¬p\neg{p}¬p
    • NOT
  • Compound statements, using the following logical connectives
    • Conjunction ∧\wedge∧
      • AND
    • Disjunction ∨\vee∨
      • OR
      • p∨qp\vee{q}p∨q means inclusive disjunction. True if one or the other of p,qp, qp,q is true or if both of the statements p,qp, qp,q are true
      • p⊻qp\veebar{q}p⊻q maens exclusive disjunction. True if one or the other of p,qp, qp,q is true but not both of the statements p,qp, qp,q are true
    • Implication →\to→
      • THEN
      • p→qp\to{q}p→q
      • If ppp, then qqq
      • ppp is sufficient for qqq
      • ppp is a sufficient condition for qqq
      • qqq is necessary condition for ppp
      • qqq is a necessary condition for ppp
      • ppp only if qqq
      • The statement ppp is called the hypothesis of the implication; qqq is called the conclusion.
    • Biconditional ↔or⇔\leftrightarrow\text{or}\Leftrightarrow↔or⇔
      • IF AND ONLY IF
      • IFF
      • p↔qp\leftrightarrow{q}p↔q means ppp is necessary and sufficient for qqq.

Tautology and Contradiction

  • Tautology
    • A compound statement is called a tautology if it is true for all truth value assignments for its component statement.
  • Contradiction
    • If a compound statement is false for all such assignments, then it is called a contradiction.

Logical Equivalence

Two statements s1,s2s1, s2s1,s2 are said to be logically equivalent, and we write s1⇔s2s1\Leftrightarrow s2s1⇔s2 , when the statement s1s1s1 is true (respectively, false) if and only if the statement s2s2s2 is true (respectively, false).

Modus Ponens (Rule of Detachment)

  • Statement
    • If ppp is true, and p→qp\to qp→q is true, then the conclusion qqq must also be true

Law of The Syllogism

  • Statement
    • If ppp then qqq
    • If qqq then rrr
    • 2 of the aboves imply if ppp then rrr
  • Math Expression
    • [(p→q)∧(q→r)]→(p→r)[(p\to q)\wedge(q\to r)]\to(p\to r)[(p→q)∧(q→r)]→(p→r)

Modus Tollens

  • Statement
    • If ppp then qqq, then not qqq impies not ppp
  • Math Expression
    • [(p→q)∧¬q]→¬p[(p\to q)\wedge\neg q]\to\neg p[(p→q)∧¬q]→¬p

Rule of Conjunction

  • If p,qp, qp,q are true statement, then p∧qp\wedge qp∧q is also a true statement

Set Theory

Terms Definitions

  • Set

    • A well-defined collection of objects. Neither order nor repetition is relevant for a general set

  • Element

    • Objects inside the set is called elements. They are said to be members of the set

  • Universe

    • The set that contains all possible elements relevant to a particular discussion or problem

  • Subset

    • If every element of CCC is an element of DDD, then CCC is a subset of DDD, which will be denoted by C⊆DC \subseteq DC⊆D

  • Proper subset

    • If, in addition, DDD contains an element that is not in CCC, then CCC is called a proper subset of DDD. It will be denoted by C⊂DC \subset DC⊂D

  • Cardinality

    • Also called size, the number of elements in a set, denoted by ∣A∣|A|∣A∣

  • Equal

    • CCC & DDD are said to be equal when C⊆DC \subseteq DC⊆D and D⊆CD \subseteq CD⊆C
